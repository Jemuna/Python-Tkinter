Countdown Timer - Tkinter App

A simple Countdown Timer application built using Python and Tkinter. This application allows users to input time in seconds, then starts a countdown, displaying the time remaining in MM:SS format. Once the countdown reaches zero, it shows a message saying "Time's up!".

Features:

Input Time: Enter time in seconds to start the countdown.
Real-time Countdown: Displays the remaining time in MM:SS format.
User-friendly Interface: Built with Tkinter for a simple and interactive GUI.
Error Handling: Displays an error message for invalid inputs.

Requirements:

Python 3.x
Tkinter (comes pre-installed with Python)
